## ADDED Requirements

### Requirement: Switch Builder Modes
The system MUST allow switching between manual and Agent builder modes.

#### Scenario: User switches to Agent mode
- **WHEN** the user selects Agent mode
- **THEN** the current draft and input are preserved

#### Scenario: User switches back to manual mode
- **WHEN** the user returns to manual mode
- **THEN** the selected asset type and draft remain editable
